# Deep Dives

Thư mục này giữ các tài liệu chuyên sâu từ bản gốc. Chúng vẫn hữu ích để học ý tưởng, nhưng một số chi tiết có thể đã lỗi thời theo version Claude Code hiện tại.

Khi dùng các file này:

1. Ưu tiên [../claude-code-handbook.md](../claude-code-handbook.md) cho workflow và khuyến nghị hiện tại.
2. Đối chiếu [../references/research-sources.md](../references/research-sources.md) trước khi copy command/settings/hook schema.
3. Kiểm tra official docs nếu nội dung liên quan đến model name, pricing, hook input schema, MCP server package hoặc settings field.
